# JEBC (Standalone Executable)

> Write C++ and JS/TS/TSX in a single `.jebc` file. Build to WebAssembly in one command.

This is the standalone Windows executable distribution of JEBC.

## Installation

1. Place `jebc.exe` somewhere in your PATH.
2. Ensure you have Node.js installed.
3. Run `setup.bat` to automatically install `esbuild` and `Emscripten` (emcc).

## Usage

```bash
jebc app.jebc
```

For more details, see the `docs/` folder or visit the GitHub repository.
