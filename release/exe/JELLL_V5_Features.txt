========================================================================
JELLL V5 完全機能リスト
========================================================================

JELLL（JavaScript Ecosystem Bundled Compiler）は、
WebAssemblyを使ったネイティブプログラムと、JavaScript/TypeScriptといった
モダンなWeb技術を「1つのエコシステム」として統合し、
最速でビルドするための専用コンパイラです。

V5 安定稼働版として、以下の全機能を搭載しています。

------------------------------------------------------------------------
【1】C/C++・Rust・Zig と JS/TS を1つのファイル（.jelll）に混在して書ける！
------------------------------------------------------------------------
ファイルの拡張子を `.jelll` にするだけで、
C / Rust / Zig / JS / TS / TSX 全てを同居させられます。
コンパイラが自動でネイティブ言語をWASMに変換し、
1つのWASMバイナリにリンクします！

    lang.c {
        extern "C" { int calc_c() { return 10; } }
    }
    lang.rs {
        #[no_mangle] pub extern "C" fn calc_rs() -> i32 { 20 }
    }
    lang.zig {
        export fn calc_zig() i32 { return 30; }
    }
    lang.js {
        console.log(native.calc_c() + native.calc_rs() + native.calc_zig());
    }

------------------------------------------------------------------------
【2】「jelll dev」による爆速ローカル開発環境
------------------------------------------------------------------------
ターミナルで `jelll dev app.jelll` と入力すると、
ローカルWebサーバーが立ち上がります。
エディタでコードを保存すると自動リビルド！

------------------------------------------------------------------------
【3】全自動・完全環境構築 — 対話型マルチセレクトUI
------------------------------------------------------------------------
`npm install -g jelll` の後、`jelll --setup` で対話型セットアップ起動！

    ? どのコンパイラをインストールしますか？
      [ ] C/C++ (Emscripten)
      [ ] Rust (wasm32)
      [ ] Zig
    (Space: 選択/解除, Enter: 決定)

矢印キーで移動、スペースで [●] をトグル、Enterで確定。
選択されたツールが自動ダウンロード＆インストール＆PATH登録されます。

------------------------------------------------------------------------
【4】CSS と HTML も直接埋め込める完全フルスタック
------------------------------------------------------------------------
    lang.html { <div id="app"></div> }
    lang.css { body { background: black; } }

ビルド時に自動で `index.html` に注入されます。

------------------------------------------------------------------------
【5】C構造体を TypeScript クラスに自動変換！（@jelll-sync）
------------------------------------------------------------------------
C構造体に `// @jelll-sync` を付けるだけで、
自動で TS クラス（getter/setter/メソッド完備）が生成されます！

------------------------------------------------------------------------
【6】JS/TS ファイルの完全なモジュール分離制
------------------------------------------------------------------------
`import` / `export` が完全に使えます。

------------------------------------------------------------------------
【7】テキストまるごとコピペ合体機能「call」
------------------------------------------------------------------------
`call "components/ui.jelll"` で別ファイルの内容をインライン展開。

------------------------------------------------------------------------
【8】コマンド1発ビルド
------------------------------------------------------------------------
    jelll main.jelll

Makefile も CMakeLists も Webpack も一切不要！
